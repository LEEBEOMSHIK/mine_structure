# Animations

Blockbench에서 내보낸 Bedrock animation 파일을 이 폴더에 둔다.

## 등록 파일

- `unicorn_toilet.animation.json`
  - `animation.unicorn_toilet.lid_open`
  - `animation.unicorn_toilet.lid_close`
  - `animation.unicorn_toilet.flush`

현재 파일은 `unicorn_toilet_spec.md` 기준으로 만든 연결 초안이다. Blockbench에서 실제 export한 파일이 있으면 같은 경로에 덮어쓴다.
