# Furniture Content

가구 콘텐츠는 각 항목별 문서로 관리한다.

## Registered

- `unicorn_toilet.md` — `mine_structure:unicorn_toilet`
- `unicorn_dining_table.md` — `mine_structure:unicorn_dining_table`
- `unicorn_chair.md` — `mine_structure:unicorn_chair`

## Naming

- Bedrock identifier: `mine_structure:<content_id>`
- File stem: `<content_id>`
- Texture folder: `addon/resource_pack/textures/entity/<content_id>/`
- Geometry file: `addon/resource_pack/models/entity/<content_id>.geo.json`
