# Content Registry

이 폴더는 add-on에 들어갈 가구와 무기 콘텐츠를 관리한다.

## 분류

- `furniture/` — 배치형/상호작용형 가구
- `weapons/` — 무기 아이템, 장비, 전투용 엔티티
