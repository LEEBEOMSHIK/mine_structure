# Agent Guide Index

이 폴더는 Minecraft Bedrock 패키지 제작에서 Codex와 Claude가 공통으로 참조할 작업 지침을 담는다.

## 읽는 순서

1. `environment.md` — 작업 환경, 도구, 좌표계
2. `principles.md` — 모델링과 품질 기준의 핵심 원칙
3. `blockbench-mcp-rules.md` — Blockbench/MCP 작업 시 반드시 지킬 주의점
4. `blockbench-mcp-setup.md` — Codex/Claude에서 Blockbench MCP를 등록하고 export하는 절차

## 적용 범위

- 대상: 이 저장소의 모든 Minecraft Bedrock 패키지 프로젝트
- 프로젝트별 진행 현황과 상세 스펙은 `PROJECTS.md`에서 활성 프로젝트를 확인한 뒤 해당 `projects/<project_id>/` 아래 문서를 따른다.

루트의 `AGENTS.md`와 `CLAUDE.md`는 `PROJECTS.md`와 함께 이 인덱스를 읽고, 프로젝트 작업 전 이 문서들을 확인해야 한다.
